# catwalk

> A tweet curation tools for humanities 

## Shortcuts

- `↑` — previous
- `↓` — next
- `→` — IN
- `←` — OUT
- `u` — UNDECIDED
- `s` — save

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
